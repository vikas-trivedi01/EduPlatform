1. Introduction to Web Development

Web development involves creating websites and web applications using core technologies like HTML, CSS, and JavaScript.

2. Front-end Development

  • HTML5 : Structure of web pages (e.g., forms, semantic tags like "article" and "section"
  • CSS3: Styling using properties like flexbox, grid, animations, and media queries.
  • JavaScript: Client-side scripting, DOM manipulation, event handling.

3. Back-end Development

  • Node.js & Express.js: Build REST APIs, handle requests/responses.
  • Databases: SQL (MySQL, PostgreSQL), NoSQL (MongoDB).
  • Authentication & Security: JWT, OAuth.
